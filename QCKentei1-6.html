Inspección por muestreo¶

En el control de calidad, existen varios tipos de inspecciones. Entre ellos, el muestreo de aceptacin se utiliza ampliamente. Este proceso implica tomar una muestra de un lote de productos, realizar pruebas o mediciones en ella, y luego comparar lo resultados con los criterios de aceptacin y rechao del lote para determnar si el lote es aceptado o rechazado.

06-01 Inspección por muestreo¶

1. ¿Qué es la inspección por muestreo?¶

La inspeccin por muestreo es el proceso mediante el cual se toma una muestra de un lote objetivo siguiendo un mtodo de inspeccin por muestreo previamente establecido. Luego, se realizan pruebas o mediciones en la muestra y se comparan con los ciriterios de aceptacin o rechazo del lote, para determinar si dicho lote es aceptado o rechazado.

2. Tipos de métodos de inspección por muestreo¶

  1. Clasificacin segn los datos menejados

    • Inspeccin por muestreo de datos por conteo
    • Inspeccin por muestreo de datos por medicin

  2. Clasificacin segn el nmero de mestreos

  3. Clasificacin segn el mtodo para determinar los criterios de aceptacin o rechazo

    • Muestreo Shimple o Muestreo de Una Sola Muestra
    • Muestreo Doble
    • Muestreo Mltiple
    • Muestreo Secuencial

3. Diseño previo de la inspección por muestreo¶

Para llevar a cabo una inspeccin por muetreo de aceptacin, es necesario determinar de antemano el tamao de la muestra (n) y el nmero de aceptacin (c).

  • Curva probabilidad acumlada (Culva Thorndike-Haga)

4. Curva característica operativa (OC)¶

La curva CO (Curva Caracterstica de Operacin), como se muestra en la Figra 6.3, es un grfico donde el eje horizontal representa la calidad del lote (porcentaje de artculos no conformes o nmero de articulos no conformes) y el eje vertical representa la probabilidad de aceptacin del lote. En esta curva, se grafica la probabilidad de que un lote sea aceptado bajo un plan de muestreo de aceptacin especfico (n , c).

06-02 Inspección por muestreo de tipo de conteo¶

La inspeccin por muestreo de tipo de conteo acepta el lote aprobado tal como est, pero en el caso del lote no aprobado, se realiza una seleccin total para reparar los productos defectuosos o reemplazarlos por productos conformes, asegurando que todo el lote sea conforme.

(1) Garantía de calidad lote por lote¶

  • LTPD: Lot Tolerance Percent Defective

(2) Garantía de calidad promedio después de la inspección múltiples¶

  • AOQ: Avergae Outgoing Quality
  • AOQL: Average Outgoing Quality Limit
  • Curva de Calidad Promedio de Aceptacin
  • AOQL: Lmite de Calidad Promedio de Salida

from scipy.stats import binom

# JIS Z 9002:1956 Datos de la tabla de muestro
SAMPLING_TABLE = [
    (5, 0), (7, 0), (10,0), (15, 1), (15,2), (20, 1), (20,3), (25, 4), (30, 6)
]

# Condiciones establecidas
p0 = 0.01  # Lmite superior de la tasa de unidades no conformes
           #para los lotes que se desean aceptar
p1 = 0.30  # Lmites inferiro de la tasa de unidades no conformes
           # para los lotes que se desean rechazar
alpha = 0.05  # L(p0) >= 1 - alpha
beta = 0.17   # L(p1) <= beta

# Bsqueda de n y c que cumplan las condiciones
def find_sampling_plan_from_table(p0, p1, alpha, beta, table):
    for n, c in table:
        L_p0 = binom.cdf(c, n, p0)  # Probabilidad de aceptacin para p0
        L_p1 = binom.cdf(c, n, p1)  # Probabilidad de aceptacin para p1

        if L_p0 >= 1 - alpha and L_p1 <= beta:
            return n, c
    return None, None

# Ejecucin
n, c = find_sampling_plan_from_table(p0, p1, alpha, beta, SAMPLING_TABLE)

print(f"Tamao de muestra n: {n}")
print(f"Nmero de aceptaciones c: {c}")

def get_sampling_plan(aql, lot_size, inspection_level="II"):
    # Criterios de muestreo de aceptacin basados en JIS Z 9015-1 (ISO 2859-1)
    # Datos de la tabla de especificaciones
    jis_table = pd.DataFrame({
        "Code": ["J", "K", "L"],  # Cdiogo
        "Lot Size Range": [(501, 1200), (1201, 3200), (3201, 10000)],
        "Sample Size": [80, 125, 200],
        "AQL 0.40%": [(1, 2), (1, 2), (2, 3)],
    })

    # Obtener la letra de cdigo correspondiente al tamao del lote

    ac, re = selected_row["AQL 0.40%"].values[0]
    code = selected_row["Code"].values[0]

    return sample_size, ac, re, code

from scipy.stats import norm

# Configuraci[on de parmetros
mu_0 = 100.0  # Media del lote que se desea aceptar
mu_1 = 98.0   # Media del lote que se desea rechazar
sigma = 1.0   # Desviacin estndar
alpha = 0.05  # Probabilidad del riesgo del productor
beta = 0.10   # Probabilidad de riesto del productor

# Clculo de K y K
K_alpha = norm.ppf(1 - alpha)
K_beta = norm.ppf(1 - beta)

# Clculo del valor lmite inferior de aceptacin Xbar_L
n_row = ((K_alpha + K_beta) * sigma / (mu_0 - mu_1)) ** 2
Xbar_L = mu_0 - K_alpha * (sigma / math.sqrt(n_row))

# Clculo del tamao de la muestra n
n = math.ceil(n_row)  # El tamao de la muestra se redondea hacia arriba
                      # el entero ms cercano

print(f"Valor lmite inferior de aceptacin Xbar_L: {Xbar_L:.2f}")
