12-01 Métodos de gestión¶

管理の方法

1. Mantenimiento y gestón¶

維持と管理

Gestión se refiere a todas las actividades para lograr un objetivo (trabajo) de manera continua, efectiva y eficiente.

La palabra gestión incluye ambos significados: mantenimiento y mejora.

  • Matenimiento
  • Actividades de mantenimiento del estado actual / Actividades de mantenimiento de la situación actual
  • Mejora
  • Actividades de mejora
  • PDCA es una metodología de gestión, ampliamente utilizada en actividades de mejora de la calidad y mejora de procesos, que consisten en repetir secuencialmente los procesos de Planificar (P), Hacer (D), Verificar (C) y Actuar (A).

4. Problemas y desafíos¶

問題と課題

Un problema es la brecha entre el objetivo establecido y la cula necesita ser abordada y superada.

Una tarea o desafío es la brecha entre el objetivo que se pretende establecer y la realidad, la cual requiere medidas.

La Historia QC de tipo resolución de problemas 問題解決型QCストーリー es un método de resolución de problemas empírico basado en datos. En este tipo, se enfatiza el establecimiento de hipótesis, la recopilación de datos y la determinación de la causa raíz mediante el análisis de datos.

La Historia QC de tipo logro de objetivos 課題達成型QCストーリー es un medio para promover eficazmente actividades destinadas a alcanzar un objetivo mediante la búsqueda de nuevas políticas y medios, y la creación de nuevas formas de hacer frente a trabajos nunca antes experimentados o a problemas donde la investigación de las causas de la ineficiencia no se considera efectiva.

from IPython.display import display
from PIL import Image
import gdown
import re

def get_file_id(url,out):
    # 正規表現でファイルIDを抽出
    match = re.search(r'/d/([a-zA-Z0-9_-]+)', url)

    if match:
        file_id = match.group(1)
        print("ファイルID:", file_id)
    else:
        print("ファイルIDが見つかりませんでした。")

    url = f"https://drive.google.com/uc?id={file_id}"

    output = out

    gdown.download(url, output, quiet=False)

    img = Image.open(out)
    display(img)
