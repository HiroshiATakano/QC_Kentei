12-01 Métodos de gestión¶

1. Mantenimiento y gestón¶

Gestin se refiere a todas las actividades para lograr un objetivo (trabajo) de manera continua, efectiva y eficiente.

La palabra gestin incluye ambos significados: mantenimiento y mejora.

  • Matenimiento
  • Actividades de mantenimiento del estado actual / Actividades de mantenimiento de la situacin actual
  • Mejora
  • Actividades de mejora
  • PDCA es una metodologa de gestin, ampliamente utilizada en actividades de mejora de la calidad y mejora de procesos, que consisten en repetir secuencialmente los procesos de Planificar (P), Hacer (D), Verificar (C) y Actuar (A).

2. Ciclo SDCA¶

4. Problemas y desafíos¶

Un problema es la brecha entre el objetivo establecido y la cula necesita ser abordada y superada.

Una tarea o desafo es la brecha entre el objetivo que se pretende establecer y la realidad, la cual requiere medidas.

La Historia QC de tipo resolucin de problemas QC es un mtodo de resolucin de problemas emprico basado en datos. En este tipo, se enfatiza el establecimiento de hiptesis, la recopilacin de datos y la determinacin de la causa raz mediante el anlisis de datos.

La Historia QC de tipo logro de objetivos QC es un medio para promover eficazmente actividades destinadas a alcanzar un objetivo mediante la bsqueda de nuevas polticas y medios, y la creacin de nuevas formas de hacer frente a trabajos nunca antes experimentados o a problemas donde la investigacin de las causas de la ineficiencia no se considera efectiva.

import gdown
import re

def get_file_id(url,out):
    # ID
    match = re.search(r'/d/([a-zA-Z0-9_-]+)', url)

    if match:
        file_id = match.group(1)
        print("ID:", file_id)
    else:
        print("ID")

    url = f"https://drive.google.com/uc?id={file_id}"

    output = out

    gdown.download(url, output, quiet=False)

    img = Image.open(out)
    display(img)
