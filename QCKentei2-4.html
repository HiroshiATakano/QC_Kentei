04-01 Prueba estadistica¶

1. Prueba estadistica¶

La prueba estadística consiste en verificar estadísticamente una hipótesis sobre una población objetivo, como por ememplo: ¿Se puede considerar que la media poblacional es 50.0? o ¿Ha disminuido la varianza poblacional?. Para ello, se toma una muestra de la población y, a partir de los datos obtenidos, se realiza la verificación mediante métodos estadísticos.

  • Hipótesis alternativa (H1)
  • Hipótesis nula (H0)
  • Prueba unilateral
  • Prueba bilateral
  • Error de tipo 1 (error del apresurado)
  • Nivel de significación ($ \alpha $)
  • Error de tipo II (error del distraido)

En general, existe una relación entre $ \alpha $ y $ \beta $: cuando se aumenta $ \alpha $, $ \beta $ tiende a disminuir, y cuando se reduce $ \alpha $, $ \beta $ suele aumentar. En una prueba estadística, es importante poder detectar correctamente cuando la hipótesis alternativa es verdadera, y la probabilidad de lograrlo se denomina poder estadístico $ (1-\beta) $.

La región de rechazo es el conjunto de valores del estadístico de prueba en el cual se decide rechazar la hipótesis nula.

from scipy.stats import norm

# 母平均と標準偏差
mu0 = 50        # H0: μ = μ0
mu1 = 52        # H1: μ = μ1
sigma = 3       # 標準偏差
n = 30          # サンプルサイズ
alpha = 0.05    # 有意水準

# 標本平均の標準誤差
se = sigma / np.sqrt(n)

# 両側検定 → α/2 両側
z_alpha = norm.ppf(1 - alpha / 2)  # 臨界値（z）
x_crit_right = mu0 + z_alpha * se
x_crit_left = mu0 - z_alpha * se

# x軸の範囲
x = np.linspace(mu0 - 4*se, mu1 + 4*se, 1000)

# H0とH1の下の分布
y0 = norm.pdf(x, mu0, se)
y1 = norm.pdf(x, mu1, se)

plt.figure(figsize=(10, 6))

# H0 の分布
plt.plot(x, y0, label='H₀: μ = μ₀', color='blue')
plt.fill_between(x, y0, where=(x <= x_crit_left) | (x >= x_crit_right), color='blue', alpha=0.3, label=r'Región de rechazo $(\alpha)$')

# H1 の分布
plt.plot(x, y1, label='H₁: μ = μ₁', color='red')
plt.fill_between(x, y1, where=(x >= x_crit_right), color='red', alpha=0.3)
plt.fill_between(x, y1, where=(x <= x_crit_left), color='red', alpha=0.3)
plt.fill_between(x, y1, where=(x > x_crit_left) & (x < x_crit_right), color='green', alpha=0.3, label=r'Región de aceptación $(\beta)$')

plt.axvline(x_crit_left, color='black', linestyle='--')
plt.axvline(x_crit_right, color='black', linestyle='--')
plt.text(mu0, -0.09, 'μ₀', ha='center')
plt.text(mu1, -0.09, 'μ₁', ha='center')
plt.text(x_crit_left, 0, 'valor critico', ha='right', va='bottom')
plt.text(x_crit_right, 0, 'valor critico', ha='left', va='bottom')

plt.title('Explicación de la región de rechazo y el poder en una prueba bilateral')
plt.xlabel('Media muestral')
plt.ylabel('Densidad de probabilidad')
plt.legend(loc='upper right')
plt.grid(True)
plt.tight_layout()

3. Estimación¶

La estimación consiste en inferir el valor de los parámetros de la población, como la media o la varianza, a partir de la distribución de la población objetivo. Existen dos tipos de estimación: la estimación puntual, que utiliza un solo estimador para inferir el parámetro, y la estimación por intervalos, que utiliza un intervalo para realizar la estmación. En la estimación por intervalos, el ancho del intervalo de confianza se determina según $ (1-\alpha) $.

4. Pasos de la estimación¶

Paso 1: Estimación puntual

La estimación puntual consiste en estimar parámetros como la media de la población $ \mu $ o la varianza $ \sigma^2 $, y normalmente se utilizan el estimador insesgado de la media $ \bar{x} $ y la varianza $ V $.

Paso 2: Estimación por intervalos

La estimación por intervalos es un método de estimación que utiliza un intervalo para estimar el valor del parámetro, y el ancho de se determina según el nivel de confianza. Generalmente, los niveles de confianza más comunes son el 95% o el 90%.

import scipy.stats as stats

# Parámetros conocidos
mu_0 = 10.0             # Media poblacional bajo la hipótesis nula H0
sigma2 = 0.5**2           # Varianza poblacional conocida
sigma = np.sqrt(sigma2) # Desviación estándar poblacional
n = 9                   # Tamaño de la muestra
x_bar = 11.0            # Media muestral

alpha = 0.05            # Nivel de significancia

# Paso 2:

# Cálculo del valor crítico para prueba unilateral (cola derecha)
z_crit = stats.norm.ppf(1 - alpha)
print(f"Valor crítico (α={alpha}): {z_crit:.3f}")

# Paso 3:

# Cálculo del estadístico de prueba Z
z = (x_bar - mu_0) / (sigma / np.sqrt(n))
print(f"Estadístico Z: {z:.3f}\n")

# Paso 4:

# Cálculo del valor p (p-value)
p_value = 1 - stats.norm.cdf(z)

print(f"Valor p: {p_value}")

if z > z_crit:
    print("Se rechaza la hipótesis nula. Se concluye que la resistencia ha mejorado.")
else:
    print("No se puede rechazar la hipótesis nula. No hay evidencia suficiente de mejora.")

En una línea de producción, el número de defectos (puntos defectuosos) en los LCDs de uso general fabricados era, tradiconalmente, de un promedio de 5 defectos por cada pulgada. import scipy.stats as stats

# Número promedio de defectos por pulgada en el pasado (5 defectos por pulgada)
mu_0 = 5

# Datos después de la mejora en el proceso
sample_size = 10  # 10 pulgadas
observed_defects = 20  # 20 defectos observados

# Estimación puntual (promedio de defectos por pulgada después de la mejora)
point_estimate = observed_defects / sample_size  # Defectos por pulgada

# Estimación del error estándar
# Asumimos una distribución de Poisson, y la desviación estándar es la raíz cuadrada de λ / n
std_error = (point_estimate / (sample_size))**0.5

# Realizamos una prueba t para verificar si la tasa de defectos es diferente
# Hipótesis nula H0: μ = 5 (tasa de defectos previa)
t_statistic = (point_estimate - mu_0) / std_error
df = sample_size - 1  # Grados de libertad

# Calculamos el valor p
p_value = 2 * (1 - stats.t.cdf(abs(t_statistic), df))

# Resultados
print("Estimación puntual: ", point_estimate)
print(f"Estadístico t:  {t_statistic:.3f}")
print(f"Valor p:  {p_value:.3f}")

# Cálculo del intervalo de confianza del 95%

confidence_interval = np.array((np.round(confidence_interval[0], 2), round(confidence_interval[1], 2)))
print("Intervalo de confianza al 95%: ", confidence_interval)

# Decisión de la prueba
if p_value < 0.05:
    print("La actividad de mejora ha reducido la tasa de defectos.")
else:

import scipy.stats as stats

# Datos de la fábrica A y B
# Número de paradas de línea y número de meses
A_paradas = 10
A_meses = 10
B_paradas = 21
B_meses = 15

# Estimación de la tasa de paradas por mes para A y B
tasa_A = A_paradas / A_meses
tasa_B = B_paradas / B_meses

# Test de hipótesis para comparar las tasas de paradas de línea
# H0: las tasas de paradas de A y B son iguales
# H1: las tasas de paradas de A y B son diferentes
# Usamos una prueba de Poisson para comparar dos tasas
# Calculamos el valor de z usando la diferencia de tasas

# Varianza de las tasas para Poisson
varianza_A = tasa_A / A_meses
varianza_B = tasa_B / B_meses

# Diferencia de tasas
diferencia_tasas = tasa_A - tasa_B

# Desviación estándar combinada
desviacion_estandar = (varianza_A + varianza_B) ** 0.5

# Estadístico de prueba z
z = diferencia_tasas / desviacion_estandar

# Valor p para la prueba de hipótesis
p_value = 2 * (1 - stats.norm.cdf(abs(z)))

# Imprimimos el valor z y el valor p
print(f'Estadístico de prueba z: {z:.3f}')
print(f'Valor p: {p_value:.3f}')

# Decisión según el valor p (nivel de significancia de 0.05)
if p_value < 0.05:
    print("Se rechaza la hipótesis nula: Hay diferencia entre las tasas de paradas en A y B.")
else:
    print("No se rechaza la hipótesis nula: No hay diferencia significativa entre las tasas de paradas en A y B.")
