Inspección por muestreo¶

En el control de calidad, existen muchos tipos de inspección. Entre ellos, la inspección por muestreo, que consiste en tomar muestras de un lote, realizar pruebas o mediciones, y compararlas con criterios de aceptación para determinar si un lote es aprobado o rechazado, se utiliza en muchas situaciones.

06-01 Inspección por muestreo¶

1. La inspección por muestreo es¶

La inspección por muestreo consiste en extraer muestras de un lote determinado, siguiendo un método de muestreo previamente establecido, para realizar mediciones o pruebas, y comparar los resultados con los criterios de aceptación del lote, con el fin de determinar si dicho lote es aceptado o rechazado.

La inspección por muestreo para datos atributivos es un tipo de inspección en la que los criterios de aceptación o rechazo del lote se basan en valores contables, como el número de productos no conformes o el número de no conformidades en la muestra.

La inspección por muestreo para datos variables es un tipo de inspección en la que los criterios de aceptación o rechazo del lote se basan en valores medibles, como la media o la desviación estándar obtenidas a partir de la muestra.

2. Curva OC (Curva Característica Operativo)¶

(|) Número de productos no conformes en la muestra¶

Se desea determinar la probabilidad de aceptación, $ L(p) $, al realizar un muestreo de aceptación único por atributos con un tamaño de muestra $ n=10 $ y un número de aceptación $ c=1 $, para un lote de $ N=1000 $ unidades con una proporción de no conformidades del 10%($ P=10 $%).

(2) Curva OC (Curva de Característica Operativa)¶

En la inspección por muestreo, se aprueban los lotes de buena calidad y se rechazan los lotes de mala calidad. Para un plan de muestreo determinado (n,c), al graficar la calidad del lote (porcentaje de artículos no conformes p%) en el eje horizontal y la probabilidad de que el lote sea aceptado L(p) en el eje vertical, se obtiene una curva. Esta curva se conoce como curva OC (Curva Característica de Operación).

06-02 Muestreo por atributos¶

1. Inspección por muestreo basado en criterios por atributos¶

La inspección por muestreo basada en criterios de conteo es un tipo de muestreo de aceptación diseñado para satisfacer tanto los requisitos del vendedor como los del comprador, al especificar la protección para el vendedor y la protección para el comprador.

La proteccioón para el vendedor se refiere a la probabilidad $ \alpha $ (riesgo del productor) de que un lote de buena calidad, con una proporción de no conformes $ p_0 $, sea rechazado por la inspección por muestreo, y esta probabilidad se establece en un valor pequeño y constante.

La protección para el comprador se refiere a la probabilidad $ \beta $ (riesgo del consumidor) de que un lote de mala calidad, con una proporción de no conformes $ p_1 $, sea aceptado, y esta probabilidad se establece es un valor pequeño y constante.

(1) Procedimiento de inspección¶

Paso 1: EStablecer los cirterios de calidad

Paso 2: Especificar los valores de p0 y p1

Paso 3: Formar el lote

Paso 4: Determinar el tamaño de la muestran (n) y el múmero de aceptación (c)

Paso 5: Tomar la muestra

Paso 6: Inspeccionar la muestra

Paso 7: Emitir el juicio de acepatación o rechazo

Paso 8: Procesar el lote

import scipy.stats as stats

# パラメータ
p0 = 0.005  # 良品率（生産者の立場）
p1 = 0.04   # 不良率（消費者の立場）
alpha = 0.1  # 生産者危険率
beta = 0.1   # 消費者危険率

def find_sampling_plan(p0, p1, alpha, beta, n_max=200):
    for n in range(1, n_max):
        for c in range(0, n):
            # 良品率でのロット合格確率（生産者側）
            L_p0 = stats.binom.cdf(c, n, p0)
            # 不良率でのロット合格確率（消費者側）
            L_p1 = stats.binom.cdf(c, n, p1)

            if L_p0 >= 1 - alpha and L_p1 <= beta:
                return n, c, L_p0, L_p1
    return None

# 求解
result = find_sampling_plan(p0, p1, alpha, beta)

if result:
    n, c, Lp0, Lp1 = result
    print(f"★ Método de muestreo por extracción → n = {n}, c = {c}")
    print(f"Probabilidad de aceptación cuando la tasa de productos buenos p0={p0*100:.1f}% => {Lp0:.3f}")
    print(f"Probabilidad de aceptación cuando la tasa de productos defectuosos p1={p1*100:.1f}% => {Lp1:.3f}")
else:
    print("No se encontró un método de muestreo por extracción que cumpla con las condiciones.")

from scipy.stats import norm
import numpy as np

# 条件設定
mu0 = 10.000  # 良品ロットの母平均
mu1 = 10.100  # 不良ロットの母平均
sigma = 0.100  # 母標準偏差
alpha = 0.05  # 生産者危険
beta = 0.10   # 消費者危険

# nとkを求める（理論値）
# 方法：逆算でz値を使ってkを求め、そのkがもう一方の条件も満たすかチェック

results = []

# 検索するnの範囲（現実的なサイズ）
for n in range(1, 51):
    z_alpha = norm.ppf(1 - alpha)
    z_beta = norm.ppf(beta)

    # 合格限界 k を z_alpha を使って算出
    k = mu0 + z_alpha * sigma / np.sqrt(n)

    # この k を mu1 側で使ったときの β を計算
    calc_beta = norm.cdf((k - mu1) / (sigma / np.sqrt(n)))

    # 条件を満たす場合、記録
    if calc_beta <= beta:
        results.append((n, k, calc_beta))
        break  # 最小の n が見つかったら終了

np.array(results)

ある粉体製造工場で、平均値が２００．０ｋｇ以上のロットは合格させたいが、１９７．０ｋｇ以下のロットはできるだけ不合格にしたい。今までのデータから、ロット内の標準偏差は２．４ｋｇであることが分かっている。生産者危険α＝０．０５、消費者危険β＝０．１０を満足する計量基準型抜き取り計算方式（サンプルサイズｎ、下限合格判定値）を求めよ。

En una fábrica de producción de polvo, se desea aprobar lotes con una valor promedio de 200.0Kg o más, y se sedea rechazar lotes con 197.0Kg o menos en lamedida de lo posible. A partir de datos históricos, se sabe que la desviación estándar dentro de los lotes es de 2.4Kg. Detemine un plan de muestreo por variables de aceptación-rechazo (tamaño de muestra n, valor límite de aceptación inferior) que satisfaga un riesgo del productor $ \alpha = 0.05 $ y un riesgo del consumidor $ \beta = 0.10 $.

import scipy.stats as stats
import math

# 与えられた条件
sigma = 2.4       # 標準偏差
mu0 = 200.0       # 生産者側の主張する平均
mu1 = 197.0       # 消費者側の主張する平均
alpha = 0.05      # 生産者危険
beta = 0.10       # 消費者危険

# 標準正規分布の逆関数からz値を求める
z_alpha = stats.norm.ppf(1 - alpha)
z_beta = stats.norm.ppf(1 - beta)

# サンプルサイズnの計算
n = ((z_alpha + z_beta) * sigma / (mu0 - mu1)) ** 2
n = math.ceil(n)  # 切り上げ

# 下限合格判定値kの計算
lower_limit = mu0 - z_alpha * (sigma / math.sqrt(n))

# 結果出力
print(f"サンプルサイズ n = {n}")
print(f"下限合格判定値 lower_limit = {lower_limit:.2f} kg")
