Inspección por muestreo¶

En el control de calidad, existen muchos tipos de inspeccin. Entre ellos, la inspeccin por muestreo, que consiste en tomar muestras de un lote, realizar pruebas o mediciones, y compararlas con criterios de aceptacin para determinar si un lote es aprobado o rechazado, se utiliza en muchas situaciones.

06-01 Inspección por muestreo¶

1. La inspección por muestreo es¶

La inspeccin por muestreo consiste en extraer muestras de un lote determinado, siguiendo un mtodo de muestreo previamente establecido, para realizar mediciones o pruebas, y comparar los resultados con los criterios de aceptacin del lote, con el fin de determinar si dicho lote es aceptado o rechazado.

La inspeccin por muestreo para datos atributivos es un tipo de inspeccin en la que los criterios de aceptacin o rechazo del lote se basan en valores contables, como el nmero de productos no conformes o el nmero de no conformidades en la muestra.

La inspeccin por muestreo para datos variables es un tipo de inspeccin en la que los criterios de aceptacin o rechazo del lote se basan en valores medibles, como la media o la desviacin estndar obtenidas a partir de la muestra.

2. Curva OC (Curva Característica Operativo)¶

(|) Número de productos no conformes en la muestra¶

(2) Curva OC (Curva de Característica Operativa)¶

En la inspeccin por muestreo, se aprueban los lotes de buena calidad y se rechazan los lotes de mala calidad. Para un plan de muestreo determinado (n,c), al graficar la calidad del lote (porcentaje de artculos no conformes p%) en el eje horizontal y la probabilidad de que el lote sea aceptado L(p) en el eje vertical, se obtiene una curva. Esta curva se conoce como curva OC (Curva Caracterstica de Operacin).

06-02 Muestreo por atributos¶

1. Inspección por muestreo basado en criterios por atributos¶

La inspeccin por muestreo basada en criterios de conteo es un tipo de muestreo de aceptacin diseado para satisfacer tanto los requisitos del vendedor como los del comprador, al especificar la proteccin para el vendedor y la proteccin para el comprador.

La proteccion para el vendedor se refiere a la probabilidad $ \alpha $ (riesgo del productor) de que un lote de buena calidad, con una proporcin de no conformes $ p_0 $, sea rechazado por la inspeccin por muestreo, y esta probabilidad se establece en un valor pequeo y constante.

La proteccin para el comprador se refiere a la probabilidad $ \beta $ (riesgo del consumidor) de que un lote de mala calidad, con una proporcin de no conformes $ p_1 $, sea aceptado, y esta probabilidad se establece es un valor pequeo y constante.

(1) Procedimiento de inspección¶

Paso 1: EStablecer los cirterios de calidad

Paso 2: Especificar los valores de p0 y p1

Paso 3: Formar el lote

Paso 4: Determinar el tamao de la muestran (n) y el mmero de aceptacin (c)

Paso 5: Tomar la muestra

Paso 6: Inspeccionar la muestra

Paso 7: Emitir el juicio de acepatacin o rechazo

Paso 8: Procesar el lote

import scipy.stats as stats

# 
p0 = 0.005  # 
p1 = 0.04   # 
alpha = 0.1  # 
beta = 0.1   # 

def find_sampling_plan(p0, p1, alpha, beta, n_max=200):
    for n in range(1, n_max):
        for c in range(0, n):
            # 
            L_p0 = stats.binom.cdf(c, n, p0)
            # 
            L_p1 = stats.binom.cdf(c, n, p1)

            if L_p0 >= 1 - alpha and L_p1 <= beta:
                return n, c, L_p0, L_p1
    return None

# 
result = find_sampling_plan(p0, p1, alpha, beta)

if result:
    n, c, Lp0, Lp1 = result
    print(f" Mtodo de muestreo por extraccin  n = {n}, c = {c}")
    print(f"Probabilidad de aceptacin cuando la tasa de productos buenos p0={p0*100:.1f}% => {Lp0:.3f}")
    print(f"Probabilidad de aceptacin cuando la tasa de productos defectuosos p1={p1*100:.1f}% => {Lp1:.3f}")
else:
    print("No se encontr un mtodo de muestreo por extraccin que cumpla con las condiciones.")
