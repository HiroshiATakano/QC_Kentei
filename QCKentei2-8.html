08-01 Análisis de correlación¶

1. El análisis de correlación es:¶

Cuando las mediciones emparejadas de dos tipos, $ x $ e $ y $, muestran que la media poblacional de $ y $ cambia linealmente en respuesta a los cambios en $ x $, se dice que existe una correlación entre ambas. El método estadístico para determinar la presencia de esta correlación se denomina análisis de correlación. La existencia y fuerza de una relación correlacional pueden apreciarse aproximadamente mediante un diagrama de dispersión, pero para su determinación estadística se utiliza el coeficiente de correlación r.

  • $ r = S_{xy} / \sqrt{S_{xx} S_{yy}}$

  • El coeficiente de correlación $ r $ toma valores de -1 a +1. Cuanto más se acerca a -1, más fuerte es la correlación negativa. Cuanto más se acerca a +1, más fuerte es la correlación positiva. Además, cuanto nás se aproxima a 0, más débil es la correlación, llegando a ser no correlacionado.

    • Correlación positiva
    • Correlación negativa
    • Sin correlación / Correlación nula

  • El coeficiente de correlación mide el grado en que $ x $ e $ y $ tienen una relación lineal. Si hay valores atipicos en el conjunto de datos o si la relación es curvilínea, calcular el coeficiente de correlación carece de sentido.

  • El coeficiente de correlación tiene sentido cuando tanto $ x $ como $ y $ siguen una distribución normal.

def compute_Sxx_Syy_Sxy(df, x_col='x', y_col='y'):
    df_trimmed = df.iloc[:-1]
    centered = df_trimmed[[x_col, y_col]] - df_trimmed[[x_col, y_col]].mean()
    Sxx = (centered[x_col] ** 2).sum()
    Syy = (centered[y_col] ** 2).sum()
    Sxy = (centered[x_col] * centered[y_col]).sum()
    return Sxx, Syy, Sxy

(2) Prueba de hipótesis para el coeficiente de correlación poblacional (Prueba de no correlación)¶

Paso 1: EStablecimiento del objetivo de la prueba

Se realiza una prueba bilateral para determinar si existe una relación de correlación entre $ x $ e $ y $

Paso 2: Establecimiento de la hipótesis nula H0 y la hipótesis alternativa H1.

  • $ H_0 : \rho = 0 $

  • $ H_1 : \rho \neq 0 $

Paso 3: Selección del estadístico de prueba.

  • $ t_0 = r \sqrt{n-2} / \sqrt{1-r^2}$

Paso 4: Establecimiento del nivel de significancia.

  • $ \alpha = 0.05 $

Paso 5: Determinación de la región de rechazo.

  • $ R : | t_0 | \geq t(\phi, \alpha) = t(n-2, \alpha) = t(28, 0.05) = 2.048 $

Paso 6: Cálculo del estadístico de prueba.

  • $ $ t_0 = 10.06

Paso 7: Decisión sobre el resultado de la prueba.

  • $ | t_0 | = 10.06 > t(28, 0.05) = 2.048 $

Paso 8: Conclusión.

  • La hipótesis nula H0 fue rechazada y la hipótesis alternativa H1 fue aceptada. Con un nivel de significancia del 5%, se puede decir que existe una relación de correlación entre x e y.

08-02 Análisis de regresión¶

1. Qué es una ecuación de regresión¶

Mientras que el análisis de correlación es una técnica para analizar la relación entre dos variables, el análisis de regresión tiene como objetivo estimar el cambio en una variable objetivo basándose en el valor de la otra variable.

Se llama análisis de regresión simple cuando hay una sola variable explicativa, y análisis de regresión múltiple cuando hay dos o más variables explicativas.

Se puede decir que el análisis de regresión es un método para encontrar la ecuación de relación (ecuación de regresión) entre una variable objetivo (variable dependiente) y una variable explicativa (variable independiente).

La ecuación de regresión se determina al encontrar la pendiente y la intersección de la recta de manera que la suma de los cuadrados de las diferencia entre los valores del eje vertical de los puntos en el diagrama de dispersión (valores observados) y los valores del eje vertical en la recta de regresión correspondientes a los valores del eje horizontal (valores observados ) sea la mínima posible (para todos los punto del diagram de dispersión). Este método se conoce como método de mínimos cuadrados.

2. Ecuación de regresión¶

El análisis de regresión tiene como objetivo realizar predicciones y control sobre una variable objetivo utilizando las variables explicativas.

  • $ y_i = a + b x_i + \epsilon_i $

  • Variable dependiente / Variable de respuesta / Variable objetivo

  • Variable independiente / Variable predictora / Regresor

3. Método de los mínimos cuadrados y Estimación de regresión lineal simple¶

El análisis de regresión simple es un método para encontrar a y b de manera que la suma de cuadrados de los residuos (SEC) sea mínima.

  • $ b = S_{xy} / S_{xx} $

  • $ \bar{y} -b \bar{x} = \bar{y} - (S_{xy} / S_{xx}) \bar{x} $
