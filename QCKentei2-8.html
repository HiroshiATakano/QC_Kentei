08-01 Análisis de correlación¶

1. El análisis de correlación es:¶

Cuando las mediciones emparejadas de dos tipos, $ x $ e $ y $, muestran que la media poblacional de $ y $ cambia linealmente en respuesta a los cambios en $ x $, se dice que existe una correlacin entre ambas. El mtodo estadstico para determinar la presencia de esta correlacin se denomina anlisis de correlacin. La existencia y fuerza de una relacin correlacional pueden apreciarse aproximadamente mediante un diagrama de dispersin, pero para su determinacin estadstica se utiliza el coeficiente de correlacin r.

  • $ r = S_{xy} / \sqrt{S_{xx} S_{yy}}$

  • El coeficiente de correlacin $ r $ toma valores de -1 a +1. Cuanto ms se acerca a -1, ms fuerte es la correlacin negativa. Cuanto ms se acerca a +1, ms fuerte es la correlacin positiva. Adems, cuanto ns se aproxima a 0, ms dbil es la correlacin, llegando a ser no correlacionado.

    • Correlacin positiva
    • Correlacin negativa
    • Sin correlacin / Correlacin nula

  • El coeficiente de correlacin mide el grado en que $ x $ e $ y $ tienen una relacin lineal. Si hay valores atipicos en el conjunto de datos o si la relacin es curvilnea, calcular el coeficiente de correlacin carece de sentido.

  • El coeficiente de correlacin tiene sentido cuando tanto $ x $ como $ y $ siguen una distribucin normal.

$ Sxy = \sum (x_i - \bar{x})(y_i - \bar{y}) $

def compute_Sxx_Syy_Sxy(df, x_col='x', y_col='y'):
    df_trimmed = df.iloc[:-1]
    centered = df_trimmed[[x_col, y_col]] - df_trimmed[[x_col, y_col]].mean()
    Sxx = (centered[x_col] ** 2).sum()
    Syy = (centered[y_col] ** 2).sum()
    Sxy = (centered[x_col] * centered[y_col]).sum()
    return Sxx, Syy, Sxy

(2) Prueba de hipótesis para el coeficiente de correlación poblacional (Prueba de no correlación)¶

Paso 1: EStablecimiento del objetivo de la prueba

Se realiza una prueba bilateral para determinar si existe una relacin de correlacin entre $ x $ e $ y $

Paso 2: Establecimiento de la hiptesis nula H0 y la hiptesis alternativa H1.

  • $ H_0 : \rho = 0 $

  • $ H_1 : \rho \neq 0 $

Paso 3: Seleccin del estadstico de prueba.

  • $ t_0 = r \sqrt{n-2} / \sqrt{1-r^2}$

Paso 4: Establecimiento del nivel de significancia.

  • $ \alpha = 0.05 $

Paso 5: Determinacin de la regin de rechazo.

  • $ R : | t_0 | \geq t(\phi, \alpha) = t(n-2, \alpha) = t(28, 0.05) = 2.048 $

Paso 6: Clculo del estadstico de prueba.

  • $ $ t_0 = 10.06

Paso 7: Decisin sobre el resultado de la prueba.

  • $ | t_0 | = 10.06 > t(28, 0.05) = 2.048 $

Paso 8: Conclusin.

  • La hiptesis nula H0 fue rechazada y la hiptesis alternativa H1 fue aceptada. Con un nivel de significancia del 5%, se puede decir que existe una relacin de correlacin entre x e y.

08-02 Análisis de regresión¶

1. Qué es una ecuación de regresión¶

Mientras que el anlisis de correlacin es una tcnica para analizar la relacin entre dos variables, el anlisis de regresin tiene como objetivo estimar el cambio en una variable objetivo basndose en el valor de la otra variable.

Se llama anlisis de regresin simple cuando hay una sola variable explicativa, y anlisis de regresin mltiple cuando hay dos o ms variables explicativas.

Se puede decir que el anlisis de regresin es un mtodo para encontrar la ecuacin de relacin (ecuacin de regresin) entre una variable objetivo (variable dependiente) y una variable explicativa (variable independiente).

La ecuacin de regresin se determina al encontrar la pendiente y la interseccin de la recta de manera que la suma de los cuadrados de las diferencia entre los valores del eje vertical de los puntos en el diagrama de dispersin (valores observados) y los valores del eje vertical en la recta de regresin correspondientes a los valores del eje horizontal (valores observados ) sea la mnima posible (para todos los punto del diagram de dispersin). Este mtodo se conoce como mtodo de mnimos cuadrados.

2. Ecuación de regresión¶

El anlisis de regresin tiene como objetivo realizar predicciones y control sobre una variable objetivo utilizando las variables explicativas.

  • $ y_i = a + b x_i + \epsilon_i $

  • Variable dependiente / Variable de respuesta / Variable objetivo

  • Variable independiente / Variable predictora / Regresor

3. Método de los mínimos cuadrados y Estimación de regresión lineal simple¶

El anlisis de regresin simple es un mtodo para encontrar a y b de manera que la suma de cuadrados de los residuos (SEC) sea mnima.

  • $ b = S_{xy} / S_{xx} $

  • $ \bar{y} -b \bar{x} = \bar{y} - (S_{xy} / S_{xx}) \bar{x} $
