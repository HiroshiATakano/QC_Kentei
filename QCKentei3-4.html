La distribucin normal se expresa como $ N(\mu, \sigma^2) $. Es decir, el centro de la distribucin est en la media poblacional $ \mu $, y la dispersin (amplitud) de la distribucin se describe mediante la desviacin estandar poblacional $ \sigma $ (o la varianza poblacional $ \sigma^2 $ ).

Todas las distribuciones, incluida la distribucion normal, tiene las siguientes propiedades.

  • La probabilidad de que ocurra cualquier valor es mayor o igual a 0.
  • La probabilidad total dentro del rango de valores posibles es 1(100%).
  • La probabilidad de que un valor est dentro de un cierto intervalo se representa como la proporcin del rea total, tomando 1(100%) como referencia.
  • La probabilidad de estar dentro del rango $ \mu \pm 1\sigma => 68.3 $ %
  • La probabilidad de estar dentro del rango $ \mu \pm 2\sigma => 95.4 $ %
  • La probabilidad de estar dentro del rango $ \mu \pm 3\sigma => 99.7 $ %

2. Distribución normal estándar¶

Cuando la variable aleatoria $ x $ sigue una distribucin normal $ N (\mu, \sigma^2) $, al transformar $ x $ en la variable aleatoria $ u $ de la siguinte manera.

$ u = \frac{x - \mu}{\sigma} $

$ u $ sigui una distribucin normal con media 0 y desviacin estndar 1. Este proceso de transformacin de $ x $ a $ u $ se llama estandarizacin (o normalizacin). Esta distribucin normal se conoce como distribucin normal estndar y se representa como $ N(0,1^2) $.

def calculate_out_of_spec_prob(mean, std_dev, lower_limit, upper_limit):
    """
    Calcular la probabilidad de estar fuera de especificacin
    """
    prob_lower = stats.norm.cdf(lower_limit, loc=mean, scale=std_dev)
    prob_upper = 1 - stats.norm.cdf(upper_limit, loc=mean, scale=std_dev)
    return prob_lower + prob_upper

def find_required_std(mean, lower_limit, upper_limit, target_prob):
    """
    Determinar la desviacin estandar para que la probabilidad de estar
    fuera de especificaciones sea menor o sigual a target_prob
    """
    z_lower = stats.norm.ppf(target_prob / 2)
    z_upper = stats.norm.ppf(1 - target_prob / 2)
    std_dev_lower = (mean - lower_limit) / abs(z_lower)
    std_dev_upper = (upper_limit - mean) / abs(z_upper)
    return min(std_dev_lower, std_dev_upper)
