05-01 Inspección (Verificación)¶

Es la actividad de realizar mediciones o pruebas en productos o servicios y compararlas con los requisitos especificados en las exigencia del cliente o en las normas, para determinar si cumplen o no con dichos requisitos.

Medición: Es la actividad de investigar y aplicar métodos y medios de medición con un propósito especifico, y utilizar los resultados para lograr el objetivo inicial.

05-02 Conforme (Producto)¶

Conforme (Producto): Es el estado (o articlo) que, tras la inspección, satisface los requisitos especificados en las exigencias del cliente o en las normas.

No conforme (Producto): Es el estado (o articulo) que, tras la inspección, no satisface los requisitos especificados en las exigencias del cliente o en las normas.

from IPython.display import display
from PIL import Image
import gdown
import re

def get_file_id(url,out):
    # 正規表現でファイルIDを抽出
    match = re.search(r'/d/([a-zA-Z0-9_-]+)', url)

    if match:
        file_id = match.group(1)
        print("ファイルID:", file_id)
    else:
        print("ファイルIDが見つかりませんでした。")

    url = f"https://drive.google.com/uc?id={file_id}"

    output = out

    gdown.download(url, output, quiet=False)

    img = Image.open(out)
    display(img)
